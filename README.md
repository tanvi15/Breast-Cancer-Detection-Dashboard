# Breast Cancer Detection Dashboard

## Overview
Machine Learning based dashboard for breast cancer diagnosis using the Wisconsin Diagnostic Dataset.

## Features
- Dataset Exploration
- Diagnosis Distribution Analysis
- Feature Analysis
- Feature Relationship Visualization
- Correlation Heatmap
- Machine Learning Model Evaluation
- Tumor Prediction Interface

## Technologies Used
- Python
- Streamlit
- Scikit-Learn
- Pandas
- NumPy
- Plotly
- Matplotlib

## Model Performance
Accuracy: 96.49%

## Project Demonstration

This project was developed as a machine learning dashboard for breast cancer diagnosis prediction using the Wisconsin Breast Cancer Dataset.

For demonstration screenshots, see the screenshots folder.
